1. Create the following tables
Create table Article_15
 with columns:
	 price->currency,	 published_on->date,	 url->string,
Create table Category
 with columns:
	 created_by->string,	 name->varchar,
Create table User
 with columns:
	 password->varchar,	 created_on->date,	 picture_url->string,
Create table Tag
 with columns:
	 priority->int,	 name->varchar,
As a result SQL queries must be created

2. Connect the tables in the following way
Tag has a many to many connection to User
User has a one to one connection to Category
Category has a one to many connection to Article_15
As a result SQL queries must be created
Write the queries from point 1 and 2 in a file called creates.sql


3. Insert
Add at least two records in each table
As a result SQL queries must be created
Write the queries from point 3 in a file called inserts.sql

4. Answer the following question
Which are the Category(s) for a given Tag

Write the queries from points 4 in a file called selects1.sql
