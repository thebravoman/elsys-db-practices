1. Create the following tables
Create table Article_47
 with columns:
	 url->string,	 price->currency,	 content->long string,
Create table Category
 with columns:
	 date_created_on->date,	 name->varchar,
Create table User
 with columns:
	 age->integer,	 description->long text,	 picture_url->string,
Create table Tag
 with columns:
	 description->varchar,	 second_priority->float,
As a result SQL queries must be created

2. Connect the tables in the following way
Category has a one to one connection to Article_47
Article_47 has a one to one connection to User
User has a many to many connection to Tag
As a result SQL queries must be created
Write the queries from point 1 and 2 in a file called creates.sql


3. Insert
Add at least two records in each table
